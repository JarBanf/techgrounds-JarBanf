### Resultaat
1. Translate the following decimal numbers into binary:  

| Decimal | Binary    |
|---------|-----------|
| 16      | 00010000  |
| 128     | 10000000  |
| 228     | 11100100  |
| 112     | 01110000  |
| 73      | 01001001  |

2. Translate the following binary numbers into decimal:

| Binary    | Decimal |
|-----------|---------|
| 1010 1010 | 170     |
| 1111 0000 | 240     |
| 1101 1011 | 219     |
| 1010 0000 | 160     |
| 0011 1010 | 58      |

3. Translate the following decimal numbers into hexadecimal:

| Decimal | Hex |
|---------|-----|
| 15      | 0F  |
| 37      | 25  |
| 246     | F6  |
| 125     | 7D  |
| 209     | D1  |
